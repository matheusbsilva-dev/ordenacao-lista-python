# ordenacao-lista-python
Programa em Python que recebe números do usuário e exibe a lista em ordem original e crescente, utilizando listas e funções de ordenação.
# Ordenação de Lista em Python

Projeto desenvolvido em Python com o objetivo de praticar lógica de programação e manipulação de listas.

## 📌 Funcionalidades
- Receber 5 números do usuário
- Armazenar os valores em uma lista
- Exibir os números na ordem digitada
- Exibir os números em ordem crescente

## 🧠 Conceitos utilizados
- Listas em Python
- Laços de repetição (`for`)
- Função `sorted()`
- Entrada e saída de dados (`input` e `print`)
- Tratamento de erros (`try/except`)
